# SimilarVocabulary
Similar Vocabulary Searcher.基于NLP的相似单词检索器.

## 单词库
### goole-10000
from:
> https://github.com/first20hours/google-10000-english

### 479k words
from:
> https://github.com/dwyl/english-words

## NLP

引用spacy NLP模块，使用训练好的:```en_core_web_lg```模型进行判断

## 使用方法
在shell中:

```
./init.sh
python getSimilar.py
```
